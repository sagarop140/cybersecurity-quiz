About This Quiz

This cybersecurity quiz features 38 specialized categories covering everything from basic concepts to advanced techniques. Each category focuses on a specific area of cybersecurity knowledge.

Categories Include:

Question Format

[
    {
        "question": "String containing the actual question",
        "options": [
            "String for option 0",
            "String for option 1",
            "String for option 2",
            "String for option 3"
        ],
        "correctAnswer": Integer (index of the correct option, zero-based),
        "explanation": "String providing explanation of the correct answer",
        "weight": Integer (weight/points assigned to this question value between 0 and 3),
        "category": "String indicating which category this question belongs to"
    }
]

Questions are stored in separate JSON files for each category, making it easy to update and maintain the quiz content.
